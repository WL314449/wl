## CSS 基础概念

### 什么是 CSS

CSS（层叠样式表）是用来控制网页**外观和布局**的语言。

- **HTML** 负责内容结构（写"这里是个标题"）
- **CSS** 负责样式呈现（让标题变成蓝色、居中、加大字体）

### CSS 三要素

| 部分 | 作用 |
|-----|------|
| 选择器 | 选中谁 |
| 属性 | 改什么 |
| 值 | 改成什么 |

### 基础语法

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

### 选择器类型

| 选择器 | 写法 | 例子 |
|-------|------|------|
| 标签选择器 | 直接写标签名 | `body {}` |
| class 选择器 | `.` 开头 | `.btn {}` |
| id 选择器 | `#` 开头 | `#highlight {}` |

## 自测题与答案

### 第1题
**问题：** 如果想让页面背景变成浅灰色，应该写什么？

**答案：**
```css
body {
  background-color: #f0f0f0;
}
```

### 第2题
**问题：** 如果 HTML 是 `<button class="btn">点我</button>`，怎么用 CSS 把按钮的文字变成白色？

**答案：**
```css
.btn {
  color: white;
}
```

### 第3题
**问题：** 怎么把页面里所有段落的字体大小改成 16px？

**答案：**
```css
p {
  font-size: 16px;
}
```

---

## 已掌握的核心概念

1. CSS = 样式语言，HTML = 结构语言
2. CSS 规则 = 选择器 + 属性 + 值
3. `body {}` 选整个页面
4. `.class名 {}` 按 class 选
5. `#id名 {}` 按 id 选
6. 颜色可用英文（gray）或十六进制（#f0f0f0）
