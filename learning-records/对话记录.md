❯ 什么是css                                                                     

● CSS（层叠样式表）是用来控制网页外观和布局的语言。

  - HTML 负责内容结构（写"这里是个标题"）
  - CSS 负责样式呈现（让标题变成蓝色、居中、加大字体）

  核心机制：

  h1 {
    color: blue;
    font-size: 24px;
  }

  意思是"把所有 h1 标题变成蓝色、24像素"。

  三个主要作用：

  1. 颜色、字体、间距 — 让页面好看
  2. 布局 — 让元素在哪一行、哪一列、靠左还是靠右
  3. 响应式 — 适配手机、平板、电脑不同屏幕

  body {
    background-color: gray;
  }

  或用英文更常见：

  body {
    background-color: #f0f0f0;
  }

  为什么是这样：

  - body = 选中整个页面
  - background-color = 要改的属性（背景色）
  - gray 或 #f0f0f0 = 改成什么颜色

  类比理解：

  HTML  = 盖房子的骨架（水泥、砖头）
  CSS   = 装修（刷什么颜色、挂什么画）
